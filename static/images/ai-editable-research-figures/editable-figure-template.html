Editable HTML Research Figure

Hybrid AI Workflow for Publication-Grade Figures

This HTML template mirrors the workflow recommended in the article. It is deliberately simple to edit by hand, easy to restyle, and easy to screenshot or convert into another visual format.

1. Source Material

  • paper excerpt
  • sketch or whiteboard
  • reference figures
  • venue style constraints
Inputs should be multimodal, not only text.

2. Prompt Compiler

Use a reasoning model to extract entities, relations, reading order, labels, and field-specific design constraints.

Outputs: image prompt, SVG, HTML, caption.

3. Image Model

Use for style exploration, premium tone, soft depth, and reference-conditioned polish.

Do not let this become the only semantic source.

4. Final Deliverables

  • SVG for PowerPoint
  • HTML for blog or docs
  • Mermaid or Typst source
  • Word-ready tables and equations
Rule: use image models for visual polish and structure-first outputs for the final editable truth.
